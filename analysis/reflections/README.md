# Reflection Convention

Moi thanh vien tu them 1 file reflection rieng trong thu muc nay.

Quy uoc ten file:
- `reflection_[ten_khong_dau].md`

Vi du:
- `reflection_viet.md`
- `reflection_hoang.md`
- `reflection_quang_minh.md`
- `reflection_b_minh.md`

Noi dung toi thieu nen co:
- pham vi cong viec da so huu
- quyet dinh ky thuat chinh
- van de gap phai va cach xu ly
- bai hoc rut ra cho lan benchmark tiep theo
